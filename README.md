🧪 Boron.AI
Your Personal Research & Study Assistant
Boron.AI is a custom-built AI designed to help with deep-dive research and academic study. It doesn't just "chat"—it uses real-time web data to provide accurate, up-to-date information for students and researchers.

📚 What can Boron.AI do?
Web-Enabled Research: It searches the live internet to find facts, dates, and recent discoveries.

Concept Simplification: It breaks down complex topics (like chemistry, history, or coding) into easy-to-understand explanations.

Study Summaries: Paste a topic, and Boron will summarize the key points you need to know.

🛠️ How it's built
Brain: Google Gemini (latest 2026 models)

Search Engine: Tavily AI (Researcher API)

Framework: Streamlit (Python)

🚀 How to Run Locally
Clone this folder.

Install the needs: pip install -r requirements.txt

Launch: streamlit run boron.py
